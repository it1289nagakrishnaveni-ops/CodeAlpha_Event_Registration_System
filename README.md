# Event Registration System - Backend Project

This project is a robust **Event Registration Backend** developed during my internship at **CodeAlpha**. It allows users to register for specific events and stores their data securely in a database.

##  Features
* **User Registration**: API endpoint to submit user and event details.
* **Database Integration**: Powered by **MongoDB** for persistent data storage.
* **RESTful API**: Clean routing and controller logic using **Express.js**.
